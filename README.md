# AI Travel Planning Agent

## Описание проекта

AI Travel Planning Agent — агентная система, которая автоматически планирует путешествие пользователя на основе его предпочтений, бюджета и ограничений.

Система использует несколько специализированных агентов и внешние API для поиска билетов, отелей и достопримечательностей, после чего формирует оптимальный маршрут поездки.

Проект демонстрирует подход к построению **агентных систем с использованием LLM**, инструментов, внешних API и памяти состояния пользователя.

---

# Проблема

Планирование путешествия требует большого количества ручных действий:

* поиск авиабилетов
* подбор отелей
* составление маршрута
* проверка бюджета
* учет погодных условий
* сравнение вариантов

Пользователь вынужден переключаться между несколькими сервисами и тратить много времени.

---

# Решение

AI Travel Planning Agent автоматически:

1. собирает требования пользователя
2. ищет варианты перелетов
3. подбирает отели
4. планирует маршрут по дням
5. оптимизирует бюджет
6. формирует итоговый travel-plan

---

# Что демонстрирует PoC

На демо система:

1. принимает параметры поездки
2. строит план действий
3. вызывает инструменты поиска
4. анализирует результаты
5. формирует итоговый маршрут

Пример:

User →
"Хочу поездку в Рим на 5 дней в июне, бюджет 1200€"

Agent →

* подбирает авиабилеты
* находит отели
* строит маршрут по дням
* выдает travel-plan

---

# Архитектура системы

Travel Planner Agent состоит из нескольких модулей:

* Planner Agent
* Flight Search Tool
* Hotel Search Tool
* Attractions Search Tool
* Budget Optimizer
* Memory Store

---

# Что НЕ входит в PoC

Out-of-scope:

* реальное бронирование билетов
* оплата
* интеграции с банковскими системами
* сложная персонализация
* поддержка всех стран и городов

---

# Используемые технологии

* Python

* LangGraph / LangChain
* External APIs
* Vector DB
* PostgreSQL / Redis
