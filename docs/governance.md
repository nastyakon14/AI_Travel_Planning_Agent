# Governance

## Risk Register

| Риск                       | Вероятность | Влияние | Детект     | Защита                 | Остаточный риск |
| -------------------------- | ----------- | ------- | ---------- | ---------------------- | --------------- |
| Галлюцинации LLM           | средняя     | среднее | evals      | ограничение источников | низкий          |
| Prompt injection           | средняя     | высокое | фильтрация | sandbox tools          | средний         |

---

# Политика логирования

Логируются:

* входной запрос пользователя
* план действий агента
* вызовы инструментов
* ошибки

Не логируются:

* персональные данные
* платежные данные

---

# Работа с персональными данными

Система хранит только:

* предпочтения пользователя
* историю запросов

Не хранит:

* паспортные данные
* платежные данные
* точные адреса

Все данные анонимизируются.

---

# Защита от Prompt Injection

Используются:

* фильтрация пользовательских инструкций
* изоляция инструментов
* whitelist инструментов

---

# Подтверждение действий

Перед критическими действиями (например бронирование):

Agent должен запросить подтверждение пользователя.

В PoC бронирование отключено.
